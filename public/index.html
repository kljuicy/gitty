Ready to Level Up Your Commits?

Get started in less than 2 minutes. No joke.

1

Install Gitty

npm install -g @kljuicy/gitty
2

Set up your AI provider

# For OpenAI (recommended)
gitty --set-key --provider openai
# Or for Google Gemini (free tier!)
gitty --set-key --provider gemini
3

Start committing like a pro

gitty

Requirements

  • Node.js 22+ (Current LTS)
  • Git installed and ready
  • API key for OpenAI or Gemini
  • A sense of humor (optional but recommended)

Pro tip: Gemini has a generous free tier, perfect for trying Gitty!
